私人秘书是一个为 SillyTavern 设计的扩展。它在你的 SillyTavern 界面中集成了一个功能完备的日历和任务管理器，让你在与角色互动的间隙也能高效管理日程。

# ✨ 功能特性

* 📅 **日历视图**：可视化的月度日历，按日期查看和管理任务。
* ✅ **任务管理**：创建、编辑和删除任务，支持设置开始和结束时间。
* ⚖️ **四象限法则**：内置基于“重要/紧急”程度的优先级系统，并配有颜色区分。
* ☁️ **TickTick 同步**：支持将当天的任务一键同步到滴答清单 (TickTick)。

# ⚠️ Notes
* 任务数据存储在 SillyTavern 的本地设置文件 (`settings.json`) 中，请定期备份。
